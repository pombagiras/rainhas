# 🌑 POMBAGIRAS.COM | Conhecimento Sagrado & Arte Visceral

> **"Lux in Tenebris, Vis in Cruce"** — Onde a tecnologia encontra a ancestralidade e as sombras revelam a verdade.

Este repositório contém o núcleo do portal educativo dedicado à desmistificação das tradições afro-brasileiras e à exaltação do feminino proibido. Desenvolvido por **Alexia Melusine**, o projeto une estética *Dark Wave*, desenvolvimento web de alta performance e espiritualidade visceral.

---

## ✨ A Essência do Projeto

O portal foi construído para ser mais do que um site; é um portal de impacto que ocupa a totalidade dos sentidos.

*   **Impacto Visual (Full Viewport):** Uma abertura cinematográfica que consome a tela inteira para imersão imediata.
*   **As 24 Falanges:** Sistema interativo de cards com expansão modal, revelando fundamentos de dia, cores, oferendas e advertências.
*   **Conhecimento Profundo:** Glossário sagrado completo e FAQ expandido para combater o preconceito religioso.
*   **Arquitetura Fluida:** Código limpo em HTML5, CSS3 (Gothic-Abyssal Design) e JavaScript puro.

---

## 🛠️ Tecnologias Utilizadas

*   **HTML5 & CSS3:** Estrutura semântica e estilização com variáveis modernas e estética de alto contraste.
*   **JavaScript (Vanilla):** Motor de interatividade para modais dinâmicos e controle de interface.
*   **Google Fonts:** Tipografia selecionada (*Cinzel* e *Montserrat*) para evocar o sagrado e o clássico.
*   **Font Awesome:** Iconografia para conexões sociais e navegação.
